#PROJECT TITLE-Report Card Generation System
#Project Overview- The Report Card Generation System is a Python-based application that takes student details and marks as input and automatically generates a formatted report card. It ensures accuracy, reduces manual effort, and provides a clear overview of student performance.
#Features-
.Student data input module
.Marks validation and processing
.Total, average, grade calculation
.Report card generation
.Error handling for invalid inputs
#Technology used-
Python 3.4
CSV module
#Installation instructions-
1. Clone the repository
   git clone <your repo link>
2. Navigate to the project folder
   cd report-card-generator
3. Install dependencies (if any)
   pip install reportlab
#To run the program -Run the program in VS Code using "Run Python File"
